# tourist-client 旅游网商城

## 项目依赖

- React
- React-router
- ReactX
- Ant-design
- Axios

## 运行

```
git clone git clone https://github.com/PokemanMaster/tourist-client.git
cd client
npm i || npm yarn
npm run start
```
## 备注

项目运行后启动在 3000 端口
项目支持移动端（推荐）和PC端
模拟了选择商品、地址到下单全部流程
项目的后端地址: https://github.com/PokemanMaster/tourist-server
